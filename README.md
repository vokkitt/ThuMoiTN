## 📧 RSVP Form Setup

Form hiện đang sử dụng Formspree.io. Để hoạt động:

1. Đăng ký tại [formspree.io](https://formspree.io)
2. Tạo form mới
3. Copy endpoint ID
4. Cập nhật `index.html` dòng 106:
```html
action="https://formspree.io/f/YOUR_ENDPOINT_ID"
```
